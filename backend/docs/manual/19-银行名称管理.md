# 银行名称管理

一、模块概述

1.1 模块归属与作用

归属：账号部→银行名称管理，为【银行卡管理】基础数据字典模块；

用于统一维护全系统所用银行 / 代收机构名称档案，自动统计该银行名下已建档的银行卡总数量（页面绑定银行：N 张），是银行验证、银行卡管理下拉「银行名称」的数据源。

1.2 数据联动规则

新增 / 编辑银行卡（银行卡管理模块）选择银行名称后，实时反向统计更新本页面【绑定银行】卡片数量；

本模块新增银行名称后，银行验证弹窗、银行卡管理新增弹窗的【银行名称】下拉选项自动同步新增数据。

二、页面功能详情

2.1 顶部功能按钮

添加（蓝色按钮）：弹窗新增银行名称档案，保存后同步全系统下拉数据源；刷新按钮刷新实时卡片统计数量。

2.2 列表字段说明

银行名称：银行 / 代收机构名称（如乐天银行、川村沙由美代收）；

状态：开关控件（启用 / 停用），停用后：银行验证、银行卡新增时下拉不再展示该银行；已绑定存量银行卡不受停用影响；

绑定银行（N 张）：系统自动统计，统计【银行卡管理】中归属当前银行名称的有效银行卡总张数，实时联动变更；

创建时间、修改时间：档案新增 / 最后编辑记录时间；

操作：编辑、删除按钮。

2.3 单行操作

编辑：修改银行名称，保存后全系统关联字段同步刷新（银行卡管理历史数据、银行验证历史单据展示名称同步更新）；

删除：

绑定银行 = 0 张（无关联银行卡）：可直接删除；

绑定银行＞0 张（已有绑定银行卡）：禁止删除，弹窗提示「当前银行已绑定银行卡，无法删除」。

2.4 分页控件

每页条数切换、总数据统计、页码跳转。
