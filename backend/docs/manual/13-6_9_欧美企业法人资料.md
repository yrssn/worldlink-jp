# 6.9、欧美企业法人资料

一、模块概述

欧美企业法人资料模块，是日联部 - 欧美地区企业店铺注册的核心数据管理模块。

核心用途：为欧美地区跨境电商店铺注册，提供企业主体信息的集中维护与校验

数据来源：日联部担当手动新增

关键规则：法人姓名必须从「客户档案」中选择，不可新增 / 修改；担当等信息随法人自动带出

二、业务背景与约束

业务场景：用于欧美平台的企业店铺注册，需完整的公司主体、法人等信息

核心约束：

法人信息必须来自「客户档案」，确保与个人信息一致，避免重复录入和数据不一致

担当信息自动关联，无需手动填写，保证责任归属清晰，数据为后续店铺注册、付款申请等模块提供支撑
