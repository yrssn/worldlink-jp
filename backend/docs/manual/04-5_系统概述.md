# 5. 系统概述

### 5.1 系统功能

进销存是一款进货 + 销售 + 库存管理工具，主要功能包括：

功能一：进（进货）销售提需求、填需求订单、入库

功能二：销（销售）开单卖货、收银、客户单、销售统计、对账

功能三：存（库存）实时查库存、盘点、出入库、预警缺货

### 5.2 系统架构

一、整体架构分层（5 层）

前端表现层：PC 管理端、移动端

接口服务层：API 接口、登录权限接口、业务接口、第三方对接接口

业务逻辑层（核心）：入库、销售、库存、财务、权限五大业务

数据持久层：数据读写、事务处理、数据缓存、数据同步

底层基础层：操作系统、数据库、服务器、运行环境、中间件

二、技术架构

后端：PHP

前端：Vue+Element UI

数据库：MySQL

运行环境：Win10/11

部署模式：B/S 架构（浏览器访问）

### 5.3 运行环境

服务器端：Linux / Windows Server

Web服务器：Nginx

脚本语言：PHP 7.4 ~ 8.1

数据库：MySQL 5.7 +

客户端：Windows 10/11 + 主流浏览器（Chrome、Edge等）
