# 客户档案

模块概述

客户档案是为已登记的合作法人维护一套完整、标准化的身份信息档案，包括联系方式、居住信息、婚姻状态、证件信息等，为后续店铺注册、合规校验、付款申请等业务提供基础数据支撑。

业务背景与来源

数据来源：新增时，必须从「日联部注册表」中选择已登记的法人姓名，禁止手动输入不存在的法人信息。自动联动「日联部注册表」的基础信息（如法人姓名、担当、建群时间、中间人），避免重复录入。

核心目的：

统一管理合作法人的详细身份资料（身份证、驾驶证、住址、联系方式等）。

支撑店铺注册、银行验证、付款申请等业务流程的信息校验。

实现法人信息的可追溯、可查询，满足合规管理要求。

业务规则与约束

保存时校验字段合法性（如文件日期有效性）。

保存后自动更新列表数据，并与日联部注册表建立关联关系。

数据唯一性：同一法人仅允许存在一份客户档案，系统自动校验，避免重复创建。

关联关系：客户档案与日联部注册表为一对一关联

日联部担当：可新增、编辑、删除自己负责的法人档案。

管理员：可查看、编辑所有档案
